# TETRIS

TETRIS for your terminal written in Go language

![](https://gist.github.com/k0kubun/d211d8ee67df64610e0c/raw/418dc89f84919057a380082e9a4289902b58fcb9/tetris.gif)

## Installation

```bash
$ go get github.com/k0kubun/tetris
```

## Play

```bash
$ tetris
```

## Keys

```
    ←: left move
    →: right move
    z: left rotate
  x,↑: right rotate
SPACE: drop
    p: pause
    q: quit
```
