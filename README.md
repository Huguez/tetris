# TETRIS

TETRIS for your terminal written in Go language

![](https://gist.github.com/k0kubun/d211d8ee67df64610e0c/raw/5fe1dbcd9a7b01f74a640475a8ad0f6f100c2259/tetris.gif)

## Installation

```bash
$ go get github.com/k0kubun/tetris
```

## Play

```bash
$ tetris
```

## Keys

```
    ←: left move
    →: right move
    z: left rotate
  x,↑: right rotate
SPACE: drop
    p: pause
    q: quit
```
