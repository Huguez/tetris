# TETRIS

TETRIS for your terminal written in Go language

![](http://pic.k0kubun.com/GKj7TYSpEITG154.gif)

## Installation

```bash
$ go get github.com/k0kubun/tetris
```

## Play

```bash
$ tetris
```

## Keys

```
    ←: left move
    →: right move
    z: left rotate
  x,↑: right rotate
SPACE: drop
    p: pause
    q: quit
```
