# TETRIS

TETRIS for your terminal written in Go language

![](http://i.gyazo.com/bfc6814661af1899bfe2816967cec819.gif)

## Installation

```bash
$ go get github.com/k0kubun/tetris
```

## Play

```bash
$ tetris
```

## Keys

```
    ←: left move
    →: right move
    z: left rotate
  x,↑: right rotate
SPACE: drop
    p: pause
    q: quit
```
